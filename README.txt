# 💼 Personal Portfolio Website

This is a basic personal portfolio website built using **HTML** and **CSS**. It is designed to help freshers and students showcase their profile, projects, and contact information in a clean and professional layout.

---

## 🔹 Features

- 🧑 About Me section
- 💻 Projects showcase
- 📫 Contact details
- 🌐 Responsive design
- 🎨 Simple and easy to customize

---

## 🔧 Technologies Used

- HTML5
- CSS3
